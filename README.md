# Customer Churn Analysis

## Overview

This project analyzes customer churn data to identify which customer groups are more likely to leave and what retention actions can reduce churn.

---

## Business Problem

The goal of this project is to turn raw data into practical business insights. The analysis focuses on identifying patterns, measuring performance, and recommending actions that can improve decision-making.

## Key Questions

- What is the overall churn rate?
- Which contract type has the highest churn?
- How does tenure relate to churn?
- Do monthly charges affect churn?
- Which customer segments need retention campaigns?
